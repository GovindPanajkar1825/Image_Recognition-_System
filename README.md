# Image Recognition System

A web application for real-time image classification using a pre-trained MobileNetV2 model.

## Features
- Upload images and get top 3 predictions
- Uses TensorFlow/Keras for image classification
- Flask web interface with image upload support

## Technologies
- Python (TensorFlow, OpenCV)
- Flask
- HTML/CSS

## Run Locally
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python app.py`
3. Open browser at `http://127.0.0.1:5000/`
4. Upload an image to see predictions
